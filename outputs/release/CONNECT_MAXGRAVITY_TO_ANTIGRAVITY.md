# دليل ربط تطبيق Maxgravity بـ Antigravity على جهاز الكمبيوتر (وضع الحساب المحلي)

يُوضح هذا الدليل بالتفصيل الخطوات اللازمة لربط تطبيق Maxgravity على iPhone بمساعد البرمجة الذاتي Antigravity على جهاز الكمبيوتر الذي يعمل بنظام Windows.

> [!NOTE]
> **لا يتطلب هذا الإصدار أي مفاتيح Gemini API أو اشتراكات AI Studio.** يعمل خادم الجسر تلقائياً بالتكامل مع تثبيت Antigravity المصادق عليه محلياً على جهاز الكمبيوتر الخاص بك، مستهلكاً حصة الاستخدام وحقوق حسابك المحلي مباشرة.

---

## 1. المتطلبات الأساسية للتشغيل

قبل البدء، تأكد من توفر المتطلبات التالية على جهاز الكمبيوتر الخاص بك:
1. **برنامج Antigravity** مثبت ومسجل الدخول إليه (مصحح ومسجل دخول بحسابك النشط) على نظام Windows.
2. **بقاء جهاز الكمبيوتر متصلاً بالإنترنت** أثناء تشغيل المهام.
3. اتصال الـ iPhone وجهاز الكمبيوتر **على نفس الشبكة المحلية (Wi-Fi)**.
4. تثبيت Node.js (إصدار 18 أو أحدث) لتشغيل الجسر.

---

## 2. التحقق من الجاهزية وإدارة الجسر (Bridge Commands)

افتح سطر الأوامر (PowerShell أو CMD) في مجلد المشروع ونفذ الخطوات التالية:

### الخطوة 1: فحص حالة النظام والجاهزية
قبل تشغيل الجسر، يمكنك التحقق من أن واجهة CLI وحساب Antigravity الخاص بك مهيأين تماماً للربط:
```powershell
cd bridge
npm run bridge:doctor
```
سيقوم هذا الأمر بفحص وتأكيد ما يلي:
- هل واجهة Antigravity CLI مثبتة وجاهزة؟
- هل حسابك المحلي متصل ومصادق عليه بنجاح؟
- هل الجسر متصل أم غير متصل؟
- تأكيد عدم الحاجة إلى أي مفاتيح API (No API key required).

### الخطوة 2: تشغيل جسر الاتصال في الخلفية
لتشغيل جسر الاتصال محلياً وبشكل دائم في الخلفية:
```powershell
npm run bridge:start
```

### الخطوة 3: التحقق من حالة التشغيل
لمعرفة ما إذا كان الجسر يعمل بنشاط ورقم المعرف الخاص به (PID):
```powershell
npm run bridge:status
```

---

## 3. خطوات الاقتران والربط باستخدام رمز الاستجابة السريعة (QR Code)

لإنشاء اتصال آمن وبدء الربط:

### الخطوة 1: توليد رمز الاقتران
نفذ الأمر التالي في سطر الأوامر على جهاز الكمبيوتر:
```powershell
npm run bridge:pair
```
سيقوم هذا الأمر ببدء الخادم (إذا لم يكن يعمل) وتوليد صفحة اقتران مؤقتة تفتح تلقائياً في متصفح الإنترنت الافتراضي لديك، لتعرض لك رمز الاستجابة السريعة (QR Code).

### الخطوة 2: مسح الرمز من تطبيق iPhone
1. افتح تطبيق **Maxgravity** على جهاز الـ iPhone الخاص بك.
2. في شاشة البدء، اضغط على **"Scan QR code"**.
3. وجه كاميرا الهاتف نحو رمز الـ QR المعروض على شاشة الكمبيوتر.
4. بمجرد المسح، ستظهر لك شاشة تأكيد تحتوي على اسم جهاز الكمبيوتر وعنوان اتصاله.
5. اضغط على **"Connect & Trust Computer"** لإتمام عملية الاقتران.

*(إذا تعذر استخدام الكاميرا، يمكنك استخدام الخيار اليدوي بإدخال عنوان الـ IP ورمز الاقتران الظاهرين على شاشة الكمبيوتر).*

---

## 4. فهم قدرات النظام (الميزات النشطة والجزئية)

يعمل تطبيق Maxgravity على تمييز حالة الاتصال والتكامل بشكل واضح:
- **وضع الحساب المحلي (Live - نشط بالكامل):** يتم إطلاق المهام وتلقي ردود الوكيل البرمجية وبدء تنفيذها وتحديث ملفات العمل بشكل مباشر ومؤمن عبر سياق العمل المحلي.
- **مزامنة محادثات سطح المكتب الحالية (Unsupported - غير مدعومة):** لا تتيح الواجهة الرسمية لـ Antigravity حالياً مزامنة أو مرآة المحادثات المفتوحة مسبقاً على تطبيق سطح المكتب بشكل مباشر. ستظهر لك الرسالة التالية في التطبيق في حال محاولة مزامنتها:
  > "Existing desktop conversations cannot currently be mirrored live through the supported Antigravity interface. Start a Maxgravity task to control it from your phone."

---

## 5. إيقاف الجسر وإلغاء وثوقية الأجهزة

- **لإيقاف خادم الجسر بالكامل:**
  ```powershell
  npm run bridge:stop
  ```
- **عرض الأجهزة الموثوقة حالياً:**
  ```powershell
  npm run bridge:devices
  ```
- **إلغاء وثوقية جهاز آيفون معين:**
  ```powershell
  npm run bridge:revoke-device <device-id>
  ```
